Privacy Policy

Last updated: January 2026

1. Information We Collect

We collect minimal data required for bot functionality: Discord User IDs, Server IDs, and Channel IDs. When using snippet features, we store the code snippets you explicitly save.

2. How We Use Data

Data is used to provide features like premium access verification, command cooldown tracking, and snippet storage. We do not sell your data to third parties.

3. AI Data Processing

When you issue an AI command, your query is sent to our AI providers (Groq/Mistral) for processing. No personal identifying information (like your Discord username) is sent with these queries.

4. Data Deletion

You can request data deletion at any time by contacting us through our support server.
